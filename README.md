# Proyecto PROVIA

El proyecto PROVIA (**Optimización del proceso de provisión de servicios/productos especializados en tienda mediante la Inteligencia Artificial**) tiene como objetivo optimizar, a través de la IA, la provisión de servicios y productos a corto plazo. 

PROVIA aborda retos como la optimización de la provisión de productos frescos tomando el caso particular de frutas y verduras. El objetivo final es la toma de decisiones inteligentes basándose en el descubrimiento de patrones comunes que ocurran a lo largo de redes comerciales, así como desarrollar modelos predictivos explicables de las necesidades de servicios/productos especializados.

Abordar el reto escalando en los niveles de analítica:

1. Analítica descriptiva, conocer las ventas/servicios requeridos de la organización.
2. Analítica diagnóstica, explicar las ventas en función de parámetros como el calendario, precios y ofertas.
3. Analítica predictiva, predecir ventas de frescos (así como necesidades de servicios) a 7 días vista en base a modelos desarrollados para el dominio.

Este repositorio incluye aquellas utilidades software desarrolladas en el marco del proyecto y liberadas como software libre para el servicio a la comunidad.
